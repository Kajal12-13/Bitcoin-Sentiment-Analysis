# Bitcoin-Sentiment-Analysis
Analysis of Bitcoin market sentiment and trader profitability using Python and data visualization.
This project analyzes the relationship between Bitcoin market sentiment and trader performance using:

- Fear & Greed Index dataset
- Hyperliquid historical trading data

The objective is to understand how trader profitability, win rate, and trading strategies behave under different market sentiments such as Fear, Greed, Extreme Fear, and Extreme Greed.

---

# Business Objective

Financial markets are heavily influenced by investor emotions and market psychology.

This project aims to answer questions such as:

- Does market sentiment affect trader profitability?
- Which market conditions produce better trading performance?
- Which strategies perform better during Fear or Greed markets?
- How does trader win rate change across different sentiments?
